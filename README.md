# catch-of-the-day
Seafood market on React.js

# Install and Run
* `git clone https://github.com/DemyanchikVadim/catch-of-the-day.git`
* `npm install`
* `gulp`
* `http://localhost:3000`

# Used technologies
- [React](https://facebook.github.io/react/)
- [Babel](https://babeljs.io/)
- [Firebase](https://www.firebase.com/)
- [Gulp](http://gulpjs.com/)
